# tokyo-web3-hackathon

## このリポジトリについて

### backend
StrapiでCMS

### frontend
blogのテンプレート。backend(strapi)からAPIで取得した記事を元にnextのSSGする

### nft-daemon
- 記事のpublish時にnextのSSGをし直す
- NFTが売買されたときに、オーナーが変わったことをStrapiに知らせる


### contracts
サイトNFTとM&Aができるコントラクト
